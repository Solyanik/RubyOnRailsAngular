# Rademade skeleton project
#### Setting up project
1. `bundle install`
2. `rake bower:install`
3. `rake rademade_admin:bower:install`
4. `rake db:create
5. `rake db:migrate`
6. `rails c`
7. `User.create(:first_name => 'test', :last_name => 'test', :email => 'test@test.com', :admin => true, :password => '1234567890')`
6. `rails s`
